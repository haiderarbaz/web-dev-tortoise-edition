## Challenge - 04: Styling Text with Font Properties

**Task:**

- Create a paragraph of text and apply various font properties such as `color`, `weight`, `size`, and style.

**Instructions:**

- Create a paragraph (`<p>`) element with the class name `styled-text`.

- Apply the following styles to the text:

  - Set the font `color` to `red`.

  - Make the text `bold`.

  - Set the `font-size` to `18px`.

  - Apply an `italic` style to the text.

## Challenge - 05: Create a Box with Padding and Margin

**Task:**

- Create a `div` element with specific `padding` and `margin` values to control the spacing around and within the box.

**Instructions:**

- Create a `div` element with the class name `box`.

- Apply the following styles to the box:

  - Set `padding` of `20px` on all sides.

  - Set a `margin` of `30px` on all sides.

  - Add a `blue` `background-color` to the box so the padding is clearly visible.

## Challenge - 06: Create a Circle Using Border Radius

**Task:**

- Create a `div` element that appears as a perfect circle using the `border-radius` property.

**Instructions:**

- Create a `div` element with the class name `circle`.

- Apply the following styles to the div:

  - Set a `width` and `height` of `100px`.

  - Use the `border-radius` property to make the div a perfect circle.

  - Add a `red` background color to make the `circle` visible.

[Code File](./challenge-06/challeneg-06.html)

## Concepts Practice

- **CSS Font Styling:** Understanding and applying font properties including `color`, `font-weight`, `font-size`, and `font-style`

- **Box Model:** Working with `padding` and `margin` to control spacing within and around elements

- **Border Radius:** Creating rounded corners and circular shapes using the `border-radius` property

- **CSS Selectors:** Using class selectors to target and style specific elements

- **Visual Design:** Applying `background-color` to enhance visual presentation and understand box dimensions
